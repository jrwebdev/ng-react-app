# ngreact-app

App for experimenting with Angular, React and ngReact

[![Sauce Test Status](https://saucelabs.com/browser-matrix/jrwebdev.svg)](https://saucelabs.com/u/jrwebdev)

## Prerequisites

```
npm install
```

## Run

```
npm start
```

## Test

```
npm test
```

## E2E Test

```
npm start
npm run test:e2e
```

## Build
```
npm run build
```
